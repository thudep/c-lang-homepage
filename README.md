# 新生 C 语言大赛

**C 语言**是一门用途广泛的计算机程序设计语言. 学习 C 语言, 既有助于同学们学习后续必修课**计算机程序设计基础**, 也可以消除学习其他编程语言的障碍.

新生 C 语言大赛由工物科协举办, 面向全校零基础新生, 以 C 语言学习与物理仿真为主题.
